# Dataframe of annual population counts by age and race, 1979-2020

A data set containing annual population counts by age (5-year bins, 18
groups), sex, and race. All data are from the US Census Bureau
Population Estimates Program.

## Usage

``` r
pop_est
```

## Format

A data frame with 9072 rows and 6 columns

- year:

  year of observation

- age:

  starting age for that age category

- age_cat:

  age category in human readable form

- sex:

  male, female, or both

- race:

  race/ethnicity group

- pop:

  number of people for year/age/race/sex combination

## Source

<https://www.census.gov/programs-surveys/popest.html>
