# Mapping of state/territory name to abbreviation to FIPS code

Mapping of state/territory name to abbreviation to FIPS code

## Usage

``` r
st_fips_map
```

## Format

A data frame with 60 rows and 3 columns

- name:

  character, name of territory

- abbrev:

  character, abbreviation

- fips:

  numeric, FIPS state code (zero-pad to 2 digits for a FIPS string)

- nchs:

  numeric, NCHS state code (zero-pad to 2 digits)
