# Is this an ICD-9 data year? (US MCOD coded causes in ICD-9 for 1979-1998)

Central definition of the ICD-9 era boundary shared by the flag\_\*
family and unite_records() so the magic numbers live in one place.

## Usage

``` r
.is_icd9(year)
```

## Arguments

- year:

  a single year

## Value

logical
